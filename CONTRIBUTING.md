# Contribuir

Las mejoras son bienvenidas.

## Propuestas útiles

- Manejo de errores con respuesta a Telegram.
- Límites por usuario.
- Lista de usuarios autorizados.
- Selección de estilos.
- Control de costes.
- Eliminación automática de mensajes temporales.
- Compatibilidad con otros proveedores de imagen.
- Mejoras de privacidad y borrado de ejecuciones.

## Proceso

1. Crea un fork.
2. Abre una rama descriptiva.
3. Realiza los cambios.
4. Comprueba que no existan credenciales.
5. Documenta el comportamiento.
6. Abre un pull request.

No incluyas fotografías personales de terceros en pruebas o incidencias.
