# Uso de las imágenes

Las imágenes del directorio `assets` se incluyen únicamente para mostrar el funcionamiento del proyecto y documentar el workflow.

La licencia MIT del repositorio se aplica al código, al workflow y a la documentación, pero no concede automáticamente derechos sobre la imagen, el rostro, la fotografía fuente o la identidad de las personas representadas.

No reutilices las imágenes de demostración para campañas, datasets, publicidad o identificación de personas sin autorización.
